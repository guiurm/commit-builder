# commit-builder

**commit-builder** is a CLI tool written in TypeScript that helps developers create Git commits that are organized, standardized, and easy to follow. It ensures consistency in commit messages by providing a structured format for your commit workflow.

## Features

- **Standardized commit messages**: Ensure your commit messages follow a specific format.
- **Easy to use**: Simple command-line interface to guide you through creating commits.
- **Customizable templates**: Set up custom templates to fit your team's workflow.
- **Built-in validation**: Ensure commits meet your format requirements before they are made.
